# API Contract Draft

Endpoint awal yang disiapkan pada service layer:

- `GET /overview`
- `GET /trainsets`
- `GET /trainsets/:id`
- `GET /cars`
- `GET /cars/:id`
- `GET /insights`
- `GET /insights/:id`
- `GET /alarms`
- `GET /maintenance`
- `GET /work-orders`
- `GET /telemetry`
- `GET /reports`

Jika endpoint final dari REKA berbeda, cukup sesuaikan file pada `frontend/src/services/`.
